# rare-minnow-brain-methylome
WGBS methylome analysis of rare minnow brain aging

This repository contains scripts used for the analysis of whole-genome bisulfite sequencing (WGBS) data from rare minnow brain samples across three age stages.

## Structure

01_data_description – dataset description  
02_processing_pipeline – WGBS processing workflow  
03_analysis_scripts – downstream analyses  
04_figures – figure generation scripts  
05_metadata – sample information  

## Software

SOAPnuke  
BSMAP  
metilene  
R 4.5.2
